# Breeze 主题

## 项目介绍

Breeze 是一个为 Hexo 博客框架开发的主题。它旨在提供一个清新、简洁且功能丰富的博客主题。

## 主要特性

1. 响应式设计：适应不同屏幕尺寸的设备
2. 文章目录（TOC）：在文章页面侧边栏显示
3. 标签云：展示博客中使用的所有标签
4. 返回顶部按钮：方便长页面的导航
5. 代码高亮：支持多种编程语言的语法高亮
6. Markdown 样式：优化的 Markdown 渲染样式

## 文件结构

- `themes/breeze/`: 主题文件夹
  - `layout/`: 布局模板
  - `source/`: 静态资源
    - `css/`: 样式文件
    - `js/`: JavaScript 文件
